# บอทพอร์ตหุ้น (เว็บแอป app.py + สคริปต์ stock_bot.py)

ดึงราคาหุ้น US จากตลาดด้วย **yfinance** (ใช้ข้อมูล Yahoo Finance ฟรี — ไม่ต้องมี API key และไม่เสียค่าใช้จ่ายเพิ่ม) แล้วคำนวณกำไรขาดทุนพอร์ต วิเคราะห์แนวโน้มราคา และประเมินความเสี่ยง

มี 2 แบบให้เลือกใช้:
- **เว็บแอป (`app.py`)** — หน้าเว็บดูง่าย เพิ่ม/ลบหุ้น คำนวณ และมีหน้าแดชบอร์ด *(แนะนำ)*
- **สคริปต์ (`stock_bot.py`)** — รันในเทอร์มินัล ออกรายงานเป็นไฟล์ Excel/HTML

---

## วิธีใช้เว็บแอป (แนะนำ)

ติดตั้งครั้งเดียว (ต้องมี Python 3.9+ ก่อน):

```
pip install flask yfinance pandas numpy openpyxl
```

รันแอป:

```
python app.py
```

แล้วเปิดเบราว์เซอร์ไปที่ **http://127.0.0.1:5000**

แอปมี 2 หน้า สลับได้จากเมนูด้านบน:

- **จัดการพอร์ต (`/`)** — พิมพ์ชื่อหุ้น + จำนวน + ราคาทุน กด "เพิ่มเข้าพอร์ต" ทำซ้ำจนครบ แล้วกด "คำนวณ & วิเคราะห์" เพื่อดูกำไรขาดทุน สัดส่วนพอร์ต แนวโน้ม และความเสี่ยง พร้อมกราฟ
- **แดชบอร์ด (`/dashboard`)** — กราฟกำไร/ขาดทุนย้อนหลัง พร้อมปุ่มเลือกช่วง **1 วัน / 7 วัน / 30 วัน / 90 วัน**, แถบความคืบหน้าสู่ **เป้าหมายมูลค่าพอร์ต** ที่ตั้งเองได้, และตารางหุ้นที่ถืออยู่ (จำนวน, ราคาเฉลี่ย, ราคาปัจจุบัน, กำไร/ขาดทุนรายตัว)

พอร์ตและเป้าหมายถูกบันทึกในไฟล์ `portfolio.csv` / `target.json` อัตโนมัติ ปิดแอปแล้วเปิดใหม่ข้อมูลยังอยู่

- หุ้นไทยใส่ `.BK` ต่อท้าย เช่น `PTT.BK`
- ถ้าเพิ่มหุ้นที่มีอยู่แล้ว ระบบจะอัปเดตจำนวน/ราคาทุนให้

## หรือใช้สคริปต์แบบเทอร์มินัล (stock_bot.py)

แก้ไฟล์ `portfolio.csv` ให้เป็นหุ้นจริงของคุณ:

```
ticker,shares,cost_basis
AAPL,10,180.50
MSFT,5,390.00
```

แล้วรัน:

```
python stock_bot.py
```

จะพิมพ์รายงานในหน้าจอ และบันทึก `report.xlsx` + `report.html`

ออปชันเพิ่มเติม:

```
python stock_bot.py --period 2y          # ข้อมูลย้อนหลัง 2 ปี (default 1y)
python stock_bot.py --rf 0.045           # อัตราปลอดความเสี่ยง 4.5%
python stock_bot.py --portfolio my.csv   # ใช้ไฟล์พอร์ตอื่น
python stock_bot.py --no-excel --no-html # พิมพ์อย่างเดียว ไม่บันทึกไฟล์
```

## รายงาน/แดชบอร์ดมีอะไรบ้าง

1. **กำไรขาดทุน (P&L)** — ราคาปัจจุบัน, มูลค่าตลาด, ต้นทุน, กำไร/ขาดทุนรายตัวและรวมพอร์ต, น้ำหนัก %
2. **แนวโน้มราคา (Trend)** — SMA20/50/200, RSI14, MACD พร้อมสัญญาณ ขาขึ้น/ขาลง/ออกข้าง
3. **ความเสี่ยง** — ผลตอบแทนรายปี, ความผันผวน, Max Drawdown, Sharpe ratio (รายตัวและระดับพอร์ต)
4. **ประวัติย้อนหลัง (แดชบอร์ด)** — เส้นกำไร/ขาดทุนตามช่วง 1/7/30/90 วัน + แถบเป้าหมาย

## หมายเหตุ

- ประวัติย้อนหลังคำนวณจาก "จำนวนหุ้นที่ถือปัจจุบัน × ราคาในอดีต" เทียบกับต้นทุนรวม (ยังไม่รวมการซื้อ/ขายระหว่างทาง)
- ข้อมูลจาก Yahoo Finance อาจดีเลย์ ~15 นาที และเป็นข้อมูลเพื่อการศึกษาเท่านั้น ไม่ใช่คำแนะนำการลงทุน
- ทั้งเว็บแอปและสคริปต์ใช้ไฟล์ `portfolio.csv` ร่วมกัน
